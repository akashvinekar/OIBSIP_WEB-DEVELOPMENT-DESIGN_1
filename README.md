# OIBSIP_WEB-DEVELOPMENT-DESIGN_1
# 🍽️ Restaurant Landing Page

## Overview

This project is a responsive Restaurant Landing Page developed using HTML and CSS. The website provides an attractive interface for showcasing a restaurant's menu, food gallery, customer reviews, and contact information. The design focuses on user experience, modern aesthetics, and responsiveness across different devices.

---

## Features

* Responsive Navigation Bar
* Hero Section with Background Image
* Featured Menu Items
* Food Gallery Section
* Customer Reviews Section
* About Us Section
* Contact Information
* Social Media Links
* Hover Effects and Animations
* Mobile-Friendly Design

---

## Technologies Used

* HTML5
* CSS3
